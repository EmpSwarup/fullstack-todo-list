# Full stack todolist app

A fullstack todolist app made using Next JS and SQL for database.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Used

- NextJS
- SQL
- Remult
- Next Auth
